Strim

Strim is a powerful tool that allows users to trim Strava runs and edit distance metadata, automatically deleting the inaccurate activity and reuploading the trimmed one, using the Strava API.

Key Features:

  • Simple GUI for activity trimming
  • Auto-recalculation of pace and other metrics
  • OAuth integration with Strava
  • Batch processing of multiple activities
